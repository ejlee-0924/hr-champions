# HR Champions 프레임워크 개요

워크샵 오프닝 또는 처음 사용자에게 Ulrich 4역할 모델을 소개할 때 사용.

---

## FRAMEWORK

### Deliverables vs Doables

HR의 가치는 **"What I do"가 아니라 "What I deliver"**로 측정한다.

| Doables (활동) | Deliverables (산출물) |
|---|---|
| 채용 프로세스 운영 | 전략 실행에 필요한 인재 공급 |
| 교육 프로그램 설계 | 직원 역량·몰입도 향상 |
| 급여 처리·평가 시스템 | 효율적이고 공정한 보상 인프라 |
| 변화 관리 워크샵 | 조직의 변화 적응 역량 |

> "Deliverables가 정의되어야 비로소 역할과 활동도 규정된다" — Ulrich (1997)

### 4역할 2x2 매트릭스

```
              전략 (미래)
                │
  Strategic     │     Change
  Partner       │     Agent
  (전략 실행)    │     (변혁 역량)
                │
 프로세스 ──────┼────── 사람
                │
  Admin         │     Employee
  Expert        │     Champion
  (효율 인프라)  │     (직원 몰입)
                │
              운영 (일상)
```

| 역할 | Deliverable | 비유 | 핵심 활동 |
|------|-------------|------|----------|
| **Strategic Partner** | 전략 실행 | 전략 번역가 | 조직 진단, HR-전략 정렬 |
| **Administrative Expert** | 효율적 인프라 | HR 공장장 | 프로세스 리엔지니어링, Shared Services |
| **Employee Champion** | 직원 몰입·역량 | 직원 대변인 | 경청, 자원 제공, 역량 개발 |
| **Change Agent** | 변혁 역량 | 조직 촉매제 | 변화 설계, 문화 변혁, 저항 관리 |

### 왜 지금 이 프레임워크인가

1997년 프레임워크가 AI 시대에 더 유효한 이유:
- **Admin Expert의 리엔지니어링 대상이 이제 AI** — 수작업 HR 프로세스를 AI로 자동화
- **Employee Champion의 경청 채널이 확장** — Slack, 설문, 메일 데이터를 AI로 통합 분석
- **Change Agent의 다중 관점 분석이 가능** — AI로 경영진/직원/HR 관점 동시 시뮬레이션
- **Strategic Partner의 조직 진단이 실시간화** — AI가 데이터를 즉시 프레임워크에 대입

### HR 신화 vs 현실

| 신화 | 현실 |
|------|------|
| 사람 좋아해서 HR 온다 | HR은 비즈니스 목적에 맞는 공정한 조직 운영 |
| 누구든 HR 할 수 있다 | HR 활동은 비즈니스 성과에 직결 |
| HR은 비용 부서 | HR 영향력은 사업 성과 기여로 측정 |
| HR은 경찰/규제 | HR은 지적자본의 가치를 높이는 역할 |
| HR은 직원 편 | HR은 관리자+직원 양측의 챔피언 |

### 4역할 간 역설

- **Strategic Partner vs Employee Champion:** 경영 파트너 ↔ 직원 대변인 양립
- **Change Agent vs Administrative Expert:** 혁신 주도 ↔ 안정적 인프라 운영 양립

> 이 긴장을 이해하고 균형 잡는 것이 HR Champions의 핵심
