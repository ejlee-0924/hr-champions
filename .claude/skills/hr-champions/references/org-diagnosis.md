# 조직 진단 (Organization Architecture Diagnosis)

"우리 조직의 전략 실행력은 어디가 강하고 어디가 약한가?"를 진단하는 도구.

---

## FRAMEWORK

Ulrich의 조직 아키텍처 6요소 (Ch3, Table 3-4).

Strategic Partner 역할의 핵심 도구 — 조직이 전략을 실행할 수 있는 역량을 갖추었는지 6가지 요소로 진단한다.

| 요소 | 핵심 질문 |
|------|----------|
| **Shared Mindset** | 올바른 공유 문화/정체성이 있는가? |
| **Competence** | 전략 달성 필요 역량(지식·기술·능력)을 갖추었는가? |
| **Consequence** | 올바른 성과관리(측정·보상·인센티브)가 있는가? |
| **Governance** | 올바른 조직구조·커뮤니케이션·정책이 있는가? |
| **Work Process/Change** | 프로세스 개선·변화·학습 능력이 있는가? |
| **Leadership** | 필요한 리더십이 갖추어져 있는가? |

### HR 이니셔티브 우선순위 매트릭스 (Figure 3-1)

진단 후 개선 이니셔티브를 선정할 때:

```
        Impact 높음
           │
  3순위     │    1순위 (★ 우선)
  (고임팩트   │    (고임팩트
   저실행)   │     고실행)
           │
──────────┼──────────
           │
  4순위     │    2순위
  (저임팩트   │    (저임팩트
   저실행)   │     고실행)
           │
        Impact 낮음
  구현 어려움 ←──────→ 구현 쉬움
```

- **Impact:** 전략 정렬도 + 통합도 + 고객 집중도
- **Implementability:** 자원 가용성 + 시간
- 24개월 타임라인 기준 4~5개 이니셔티브만 선정

---

## EXECUTE

### 입력 요청

사용자에게 아래 정보를 요청한다:

1. **조직 개요** — 업종, 규모, 핵심 사업
2. **전략 방향** — 현재 가장 중요한 전략적 목표 2~3개
3. **현재 이슈** — 조직이 겪고 있는 주요 도전
4. (선택) 각 요소별 1~10점 자가 평가

### 분석 내용

1. **6요소별 진단** — 각 요소 현재 수준 추정 + 근거
2. **취약 영역 식별** — 전략 실행을 가장 저해하는 요소
3. **요소 간 연결 분석** — 예: Competence 부족이 Leadership 약화로 이어지는 연쇄 효과
4. **우선순위 매트릭스** — 개선 이니셔티브를 Impact x Implementability로 분류
5. **Top 3 권고 액션** — 24개월 내 실행 가능한 우선순위

## CC-CONCEPT

**연결 개념:** CLAUDE.md

> "Strategic Partner는 비즈니스 맥락을 깊이 이해해야 합니다. Claude Code도 마찬가지예요. 여러분 회사를 모르는 CC에게 전략적 맥락을 알려주는 파일이 CLAUDE.md입니다."

**CLAUDE.md란:**
- 프로젝트 루트에 두는 마크다운 파일
- CC가 대화를 시작할 때 자동으로 읽어서 맥락을 파악
- 회사 개요, 현재 이슈, 전략 방향, 팀 구조 등을 담음

**실습 안내:**

> 방금 진단에 입력한 조직 정보를 CLAUDE.md 파일로 저장해보세요. 이렇게 하면 앞으로 CC에게 매번 설명하지 않아도 우리 회사 맥락을 알고 답해줍니다.

```markdown
# CLAUDE.md

## 회사 개요
[업종, 규모, 핵심 사업]

## 현재 전략 방향
[주요 목표 2~3개]

## 조직 이슈
[현재 겪고 있는 도전]

## HR 방향
[HR이 집중해야 할 영역]
```
