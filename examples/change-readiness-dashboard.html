충돌 지점 심층 분석

Creating Shared Need 55p 차이 — 최대 충돌
경영진 (80)
"AI 안 하면 경쟁에서 뒤처진다. 시장 데이터, 경쟁사 동향으로 필요성 확신."
직원 (25)
"또 새로운 걸 배우라고? 지금도 바쁜데. 왜 필요한지 설명 한 번 못 들음."
경영진은 "왜"를 알지만 전달하지 않았음. 10C Communication(2.0) 최하위와 정확히 일치하는 패턴.
Shaping a Vision 50p 차이
경영진 (70)
"AI-First 전환으로 스마트팩토리 + 업무 자동화. 중장기 로드맵 있음."
직원 (20)
"AI가 뭘 바꾸는 건지 구체적으로 모름. 내 업무가 어떻게 바뀌는데?"
비전은 있지만 추상적. "나한테 뭐가 달라지는데?"에 대한 답이 없음. 개인 수준 비전 번역 필요.
Mobilizing Commitment 45p 차이
경영진 (60)
"CEO 직접 선언, 1억 예산 확보. 스폰서십 강력."
직원 (15)
"중간 관리자: '내 일자리 위협'. 노조 반대. 적극 저항 + 무관심."
Top 의지 ≠ 현장 헌신. 중간 관리자가 "전달자"가 아닌 "저항자". 이 층을 전환해야 변화 전파.

추천 접근법

Top-Down
경영진 → 하향 전파. 이미 CEO 선언했지만 효과 없음. 하향식 메시지가 중간에서 왜곡.
현재 상태 — 효과 없음
Side-to-Side ✅
부서 간 경계를 넘는 AI 성공 사례 횡적 확산. IT팀 → 인접 부서로 자연스럽게.
추천 — 파일럿 성공 확산
Bottom-Up ✅
현장 직원이 "우리 업무에 AI를 어떻게 쓸지" 직접 제안. GE Workout 스타일.
추천 — 현장 주도 변화

권고 액션 타임라인

1순위 — 즉시
Creating Shared Need 확보
대상: Shared Need 25→60 (직원 관점)
→ 현업 팀장이 "왜 필요한지" 자기 팀 언어로 설명
→ 구체적 벤치마크: "B사는 AI로 채용 5일 단축"
→ 직원 언어: "여러분의 야근이 줄어듭니다"
→ CEO 언어("경쟁력 강화") ❌
2순위 — 2주 내
Monitoring Progress 설계
대상: Monitoring 20→60 (전 관점)
→ 성공 지표 합의: 시간 절감, 만족도, 활용 빈도
→ 월간 대시보드 가시화
→ "진짜 효과가 있구나" 체감 설계
3순위 — 1개월 내
중간 관리자 전환
대상: Commitment 15→50 (직원 관점)
→ "위협 대상" → "AI 활용 챔피언"으로 리프레이밍
→ 관리자 전용 AI 워크샵 (프로세스 개선 실습)
→ "당신 팀 생산성을 높이는 도구"로 포지셔닝
