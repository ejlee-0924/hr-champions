구현 타임라인

Phase 1: 즉시 (1~2주)
AI 스크리닝 + 메일 자동화
→ CC로 이력서 배치 분석 Skill 생성
→ 합격/불합격 메일 템플릿 세팅
→ R&D센터 채용 3건으로 파일럿
Phase 2: 1개월 내
캘린더 연동 + 온보딩 패키지
→ Calendly 또는 Google Calendar 연동
→ 온보딩 10종 서류 디지털 템플릿
→ 전자서명 도입
Phase 3: 3개월 내
전사 확대 + 측정
→ 전 직군 채용에 확대 적용
→ 채용 KPI 대시보드 구축
→ 분기별 임팩트 리뷰
