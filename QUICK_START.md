# 5분 시작 가이드

## 1. 설치 (2분)

Claude Code Settings → Skills → 아래 6개 .skill 파일 업로드

**최소 설치:** 2개만 (`hr-role-checkup` + 관심 있는 도구 1개)

## 2. 테스트 (1분)

```
내 HR 역할 진단해줘
```

## 3. 실전 활용

```
"우리 조직 진단해줘"
"채용 프로세스 개선해줘"
"직원 몰입 분석해줘"
"AI 도입 변화관리 점검해줘"
"우리 HR팀 자가진단해줘"
```

## 추천 순서

```
1. hr-role-checkup  →  "나는 어떤 역할이 약한가?"
2. 약한 역할의 도구  →  바로 실무에 적용
```

| 약한 역할 | 쓸 도구 |
|----------|---------|
| Strategic Partner | org-diagnosis |
| Admin Expert | process-reengineering |
| Employee Champion | employee-pulse |
| Change Agent | change-readiness |

---

## 이런 상황에서 쓰세요

### "내일 경영진 보고인데 데이터가 없다"

```
"조직 진단해줘" → 6요소 분석 → 슬라이드로 출력
```
막연한 "조직이 안 좋아요"가 → "6요소 중 이 2개가 병목, 이 3개 액션 제안"으로.

### "퇴사자가 또 나왔는데 원인을 모르겠다"

```
"직원 분석해줘" → 퇴사 면담 + 설문 붙여넣기 → 10C 분석
```
텍스트 면담 기록이 → 구조화된 근본 원인 + 클러스터 진단 + 액션으로.

### "AI 도입하라는데 어디서부터?"

```
"변화관리 점검해줘" → AI 도입 특화 렌즈 자동 적용
```
"AI 도입해"라는 막연한 지시가 → 7요인 진단 + 3관점 충돌 분석 + 실패 패턴 경고로.

### "채용이 너무 느려요"

```
"프로세스 개선해줘" → 6단계 분석 → "자동화 실행할까요?" → Skill 생성
```
"빨리 해주세요" 불만이 → As-Is/Should-Be 비교 + ROI + 실제 자동화까지 한 번에.

### "우리 HR팀이 전략적이지 못하다는 피드백"

```
"HR 자가진단해줘" → 6요소 + 역량 3도메인 + AI 준비도
```
"행정만 한다" 피드백이 → 자가 진단 데이터 + "이렇게 바꾸겠습니다" 90일 플랜으로.

### "내가 HR로서 잘하고 있는 걸까?"

```
"내 역할 진단해줘" → 40문항 → 강점/개발 방향
```
막연한 불안이 → "당신은 EC형 HR, Admin Expert를 키우면 더 강해집니다"로.

## 사용 패턴

| 패턴 | 설명 | 예시 |
|------|------|------|
| **단발 소환** | 특정 상황에서 1개 도구 | 내일 보고, 퇴사 분석, 프로세스 개선 |
| **연쇄 소환** | 하나의 결과가 다음 도구 트리거 | 역할 진단 → 약한 역할 도구로 이동 |
| **풀사이클** | 6개 전체를 체계적으로 | 연간 HR 전략 수립, M&A 조직 통합 |

### 풀사이클 예시: 연간 HR 전략 수립

```
1. hr-for-hr        → "우리 HR팀 현재 수준은?"
2. hr-role-checkup  → "HR 리더들의 역할 균형은?"
3. org-diagnosis    → "조직이 전략을 실행할 준비는?"
4. employee-pulse   → "직원이 진짜 필요한 건?"
5. change-readiness → "내년 핵심 변화 프로젝트 준비도는?"
6. process-reengineering → "가장 비효율적인 프로세스 Top 3는?"
→ 6개 결과 종합 = 연간 HR 전략 보고서
```

모든 도구는 마크다운 리포트를 자동 생성한 후, 추가 시각화를 선택할 수 있습니다:

```
1) HTML 대시보드 — 인터랙티브 차트, 탭 전환, 호버 상세 (혼자 분석용)
2) HTML 슬라이드 — 화살표로 넘기는 발표용 (경영진 보고용)
3) HTML 1-pager — 한 장 요약 (빠른 의사결정용)
```
